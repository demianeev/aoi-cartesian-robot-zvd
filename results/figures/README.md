# Results Figures

This folder contains figures generated from the experimental data and simulations.

| File | Description |
|------|-------------|
| `fig_ringdown_3panel.png` | Representative post-stop ring-down for z_min, z_mid, z_max configurations |
| `fig_robustness_scatter.png` | ZVD shaper effectiveness vs. frequency ratio across all 7 configurations |
| `experimental_ringdown_y14p5_zmin.png` | Single-configuration detail: y=14.5 cm, z_min, Trial 1 |
| `fig_ringdown_zmax_y14p5.png` | Single-configuration detail: y=14.5 cm, z_max |
| `fig1_sim_ringdown.png` | Simulated ring-down from MATLAB (fn=11.1 Hz, ζ=0.035) |

Figures are regenerated automatically when you run `python/process_results.py` with your own CSV data.
